From User Interface to Agent Interface:
Efficiency Optimization of UI Representations for LLM Agents

Method Overview

Key Points

  • Problem: UI representations consume up to 80–99% of LLM agent tokens, creating a major efficiency bottleneck.
  • Goal: Automatically optimize UI structures to reduce token usage while preserving task-relevant semantics.
  • Approach: UIFormer synthesizes transformation programs using a DSL and LLM-guided iterative refinement with efficiency and correctness rewards.
  • Integration: Lightweight plugin with minimal changes to existing LLM agents.
  • Results: 48.7–55.8% token reduction across 3 benchmarks and 5 LLMs, with minimal overhead and maintained/improved performance.
  • Deployment: Successfully applied in real-world industry systems.
